# GameLibrary
Strona internetowa, na której administracja może dodawać i usuwać gry, a użytkownicy mogą je dodawać i usuwać z ulubionych.

Technologie użyte:
- Frontend: React
- Backend: Express
- Baza danych: Postgres
- AutoryzacjaL Keycloak


# Jak uruchomić
1. Sklonuj repozytorium
```git clone https://github.com/witold-kwidzinski-ug/wkwi-bezpieczenstwo-projekt.git```
2. Uruchom plik docker-compose.yaml
```docker compose up -d```

Strona znajduje się pod portem :80
